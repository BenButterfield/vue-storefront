# Events List

To keep track and make debugging of $bus.$emit events across components easier, here is a list of such events that are triggered by compontnes of default theme.

# ForgotPass

On component close:

  • modal-hide, modal-signup

On send email action:

  • notification-progress-start
  • notification-progress-stop

On error handler of email send action:

  • notification-progress-stop

# OrderConfirmation

On mounted lifecycle hook:

  • modal-show, modal-order-confirmation

On order confirmation:

  • modal-hide, modal-order-confirmation

On order cancelling:

  • modal-show, modal-order-confirmation

# OrderReview

On 'Term and conditions' link click:

  • modal-toggle, modal-terms

# PersonalDetails

On 'Term and conditions' link click:

  • modal-toggle, modal-terms

# Newsletter

On newsletter popup show:

  • modal-show, modal-newsletter

# Header

On 'Login to your account' link click:

  • modal-toggle, modal-signup

# Reviews

On 'Login to your account' link click:

  • modal-show, modal-signup

# SidebarMenu

On 'Login to your account' link click:

  • modal-show, modal-signup

# SubCategory

On user logout:

  • user-before-logout

# Language

On mounted lifecycle hook:

  • modal-show, modal-switcher

On component close:

  • modal-hide, modal-switcher

# LanguageSwitcher

On showing language popup:

  • modal-show, modal-switcher

# NewsletterPopup

On showing newsletter popup:

  • modal-show, modal-newsletter

On hiding newsletter popup:

  • modal-hide, modal-newsletter

# Onboard

On component close:

  • modal-hide, modal-onboard

# Home

On beforeMount lifecycle hook:

  • modal-toggle, modal-onboard
